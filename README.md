# serenity-mern
This is a full stack rehabilitation management system to be deployed as a b2b saas project. It has a landing page, blog and gallery.

# installation
- git clone/pull
- cd client/ && cd server -- --npm init -y/npm install
- npm run dev (to install "start" script)

# dev
- build server
- client 

```bash
# follow vite project instructions from tailwind.css
cd client
npm init -y
npm install
# install dependencies ie react, react-router
npm run build
npm run dev
```

# CI/CD pipeline
- client && server - HEROKU
- client (VERCEL) && server (RAILWAY)
